# PracticalQuiz2 - Garden Gazebo Scene

This is my submission for the online practical quiz.

## Description
A simple A-Frame 3D scene featuring:
- Gazebo with roof and pillars
- Table with Milo tin, Milk tin, and two cups
- Two chairs (one tucked under the table)
- Walkway leading to the gazebo
- Flowers for decoration
- Slow camera glide animation ending in front of the table
- Keyboard/mouse controls enabled after animation for free viewing

## How to view
Open `index.html` in any modern browser (Chrome, Edge, or Firefox).  

Enjoy the scene! 🌿
